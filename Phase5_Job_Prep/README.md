## 必备 Portfolio 项目

### 项目一：多因子选股系统（必须有）

```
要求：
├── 股票池：沪深300或中证500
├── 因子数量：6-8个（价值+质量+动量+低波）
├── 回测周期：2015-2025年（覆盖牛熊周期）
├── 年化收益：超过基准5%以上
├── Sharpe：> 1.0
└── 代码：GitHub开源，README详细
```

### 项目二：配对交易策略（加分项）

```
要求：
├── 协整检验 + 选对方法
├── Kalman滤波动态对冲比例
├── z-score信号系统
└── 回测覆盖2018-2025年（极端行情）
```

### 项目三：ML因子合成（加分项）

```
要求：
├── LightGBM多因子合成
├── Purged K-Fold 时序交叉验证
├── SHAP值特征重要性分析
└── 样本内外对比
```
